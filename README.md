# Dice ROller

A simple dice rolling app

## Instructions for Build and Use

Steps to build and/or run the software:

1. Download Android Studio
2. Download a provided emulator in Android Studio or connect your phone
3. Open the project in Android Studio
4. Run from in Android Studio

Instructions for using the software:

1. Input how many dice you want to roll in the left field (Dice #)
2. Input the number of sides wanted in the right field (Side #)
3. Press the roll button
4. Scroll through results or see the total at the bottom

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:
